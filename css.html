selector:


  • basic selector *{}, * include all elements

  • element selector eg. div{}

  • id selector(unique):

    this line has id=“txt”,
    style #txt defined in head
    *#txt style all id=txt
    h3#txt only style all h3 whose id are txt

  • class selector .

    there are two classes c and c2:
    .c{}
    .c2{}
    *.c{}

principle:

  1. 优先原则： 后解析css的内容 会覆盖之前的(.c2{} 在.c1{}之后解析， 那么无论class="c1 c2" 还是 class="c2 c1", 都会按c2执行)
    1. 同一个选择： 从上往下
    2. 同一类型选择器：从上往下
    3. 不同类型选择器 从低到高解析： * < div < class < id
    4. 先外部样式 后内部样式（为主） 合并之后一起解析
    5. 内联（在element 内部定义的样式）：外部加内部解析完了之后，才解析内联样式
    6. 加了 important 的字段最后解析
  2. 继承原则： 嵌套里面的标签 拥有外部标签的某些样式
                          子元素继承父元素的属性
    1. 跟文本相关 样式 可以被继承
    2. 块级元素 宽度 不被设置时 直接继承父元素， 高度 由内容决定

组合选择器

把基本选择器通过特殊符号串在一起
  1. 分组 , eg.: div,p{} div和p有相同样式
  2. 嵌套选择器 用空格分开 div p{} 只有最近的父级是div的p使用这种样式
  3. 相邻同级别选择器 div+p{} p的同级别是div的p会使用这个样式
