element classes:
  • 块元素： 独立成一行 可以设置宽高 默认宽高100%:
  • 文字类：p h1~h6
  • 容器类：div table tr td th form ul li ol dl dt dd
  • 行元素： 不独立成行， 不可设置 内容决定
  • a img input strong em del span
  • 特殊字符： 文字 br  
嵌套规则：
  1. 块元素客气嵌套行元素:
    2. <div><input></input></div>
  2. 行元素可以嵌套行元素:
    3. <em><p></p></em>
  3. 行元素不可以嵌套块元素:
    4. <a><div></div></a>
  4. 文字类块元素不可以快元素:
    5. <p></p><p></p>
